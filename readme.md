# Play API Project

This is a Scala Play Framework project for file handling APIs.

## How to Run
- Clone the repo
- Use `sbt run` to start the server at `localhost:9000`
- To send data use POST method and header in json format in `POST` method with url  `localhost:9000/file`
- To get data use `GET` method  `localhost:9000/file`
- To delete the file `DELETE` method  `localhost:9000/file`
